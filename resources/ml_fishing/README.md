simple fishing script get a rod and go fish 
what you do with your fish is up to you .

# Installation
1. Clone this repository.
2. Extract the zip.
3. Put ml_fishing to your resource folder.
4. Add "start ml_fishing" in your "server.cfg".

# dependency 
redem_roleplay
