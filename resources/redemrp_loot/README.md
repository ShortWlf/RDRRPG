# redemrp_loot
Closest to native rdr3 looting as possible made for RedM using redem_roleplay

# Installation
1. Clone this repository.
2. Extract the zip.
3. Put redemrp_loot to your resource folder.
4. Add "start redemrp_loot" in your "server.cfg".
5. Profit

# Required resource
- redem_roleplay
